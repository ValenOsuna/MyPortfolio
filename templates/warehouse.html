🛒 Sistema de Gestión Mayorista

Plataforma fullstack para la administración de operaciones mayoristas en supermercados. El sistema permite la gestión completa de productos, ventas, stock, clientes y cuentas de usuario. Incorpora autenticación y login seguros, con división de roles como administrador, vendedor y cliente, cada uno con su panel personalizado. La lógica de negocio incluye control de inventario en tiempo real, seguimiento de ventas y validaciones según permisos asignados.

Funcionalidades:

  • CRUD de productos, ventas, compradores, vendedores y usuarios.
  • Gestión de stock en tiempo real y control de inventario.
  • Roles diferenciados para acceso (admin, vendedor, etc.).
  • Interfaz adaptable e intuitiva.

Tecnologías utilizadas:

Backend

Flask, Docker, MySQL

Frontend

React, Bootstrap, CSS, HTML
